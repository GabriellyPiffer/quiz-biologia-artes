# 📱 Quiz de Biologia e Artes

##  Introdução do Trabalho

Este trabalho tem como objetivo o desenvolvimento de um aplicativo mobile no formato de quiz, utilizando a plataforma MIT App Inventor. A proposta é criar uma ferramenta interativa para auxiliar no aprendizado sobre os temas Biologia e Artes, tornando o estudo mais dinâmico e envolvente.
O aplicativo contém perguntas com múltiplas alternativas.
